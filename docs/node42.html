Introduction

Server / client applications are at the heart of a huge number of web applications ranging from shopping baskets to social media applications.

Generally it's easy to create dynamic content which is not realtime with many different solutions available (see PHP or nodejs) and well documented. However, feeding realtime data from C++ to a web page or realtime button presses back to C++ is more difficult.
