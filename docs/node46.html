POST (application/x-www-form-urlencoded)

When a website sends data back then even if submitted as JSON via jQuery it will be submitted in the form of: 
temperature=20&steps=100&comment=all+good%33
and needs to be parsed by the receiver. All web scripting languages such as PHP or node.js have powerful functions to decode these strings. In C/C++ libcurl can be used to convert back the escape characters to its original binary form.
