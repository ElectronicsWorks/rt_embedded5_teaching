Dealing with competition / concurrency

To avoid that two threads manipulating data at the same time one can employ a “mutex”: 

std::mutex cmdMtx;

Thread 1: 

cmdMtx.lock();
a = 1
cmdMtx.unlock();

Thread 2: 

cmdMtx.lock();
if (a = 2) { do dangerous stuff };
cmdMtx.unlock();
