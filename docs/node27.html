Lifetime of a thread

Threads terminate simply once the static thread method has finished. To tell the client that a thread has finished use another callback to trigger an event.

Sometimes it's important to wait for the termination of the thread, for example when your whole program is terminating or for example when you stop the data acquisition of the ADC. To wait for the termination of the thread use the “join()” method: 

        void stop() {
                uthread->join();
                delete uthread;
                uthread = NULL;
        }
